# 语言教学浏览器测试计划

> 创建时间：2026-06-03
>
> 目标：在浏览器端真实验证 `feat/language-teaching` 的语言教学 MVP，包括老师 persona 切换、文本流式对话、结构化纠错卡片、练习卡片和目标语 TTS 朗读。

---

## 1. 启动入口

后端：

```bash
uv run python -m asuka.main
```

前端：

```bash
pnpm --dir frontend dev
```

浏览器访问：

```text
http://localhost:5173/learn
```

前端 Vite dev server 已通过 `frontend/vite.config.ts` 把 `/ws`、`/api`、`/health` 代理到 `http://127.0.0.1:8000`。

---

## 2. 前置条件

- `.env` 中至少配置一个可用 LLM Key。默认模型为 `deepseek-chat`，需要 `DEEPSEEK_API_KEY`。
- 浏览器能访问 `http://localhost:5173/learn`。
- 页面右上角健康状态应显示后端在线。
- 当前 MVP 的朗读按钮使用 `edge-tts`，需要本机能访问其公共 TTS 端点。

---

## 3. 快速冒烟测试

### 3.1 页面与连接

1. 打开 `http://localhost:5173/learn`。
2. 确认导航栏有「语言学习」入口。
3. 确认页面包含：
   - 英语老师 / 日语老师切换。
   - 水平下拉。
   - 批改、选择题、填空题、翻译题快捷按钮。
   - 新会话按钮。
   - 输入框显示已连接状态。

通过标准：
- 页面无白屏。
- 控件不重叠。
- 输入框下方状态为「已连接」。

### 3.2 英语老师纠错

选择：

```text
英语老师
Level: A2
```

输入：

```text
帮我批改这句英语：I has a apple.
```

通过标准：
- 老师回复为流式文本。
- 回复中用中文解释问题。
- 页面出现纠错卡片。
- 纠错卡片至少包含：
  - 错误类型，例如「主谓一致」或「冠词」。
  - 地道改写，例如 `I have an apple.`
  - 英语发音标注或 IPA。
- 点击纠错卡片的朗读按钮能播放英语音频。

### 3.2.1 遇到旧会话错误时

如果页面出现类似错误：

```text
An assistant message with 'tool_calls' must be followed by tool messages...
```

处理方式：

1. 点击「新会话」按钮。
2. 等输入框状态恢复为「已连接」。
3. 重新发送测试句。

说明：该错误通常来自旧 conversation checkpoint 中残留的未配对 tool call。新会话会使用新的 `conversation_id`，不会复用旧 thread 状态。

### 3.3 日语老师纠错

选择：

```text
日语老师
Level: N5
```

输入：

```text
帮我批改这句日语：私は寿司を好きです。
```

通过标准：
- 老师用中文说明问题。
- 页面出现纠错卡片。
- 纠错卡片指出助词或表达问题。
- 地道改写应接近 `私は寿司が好きです。`
- 标注包含假名或罗马音。
- 点击朗读按钮能播放日语音频。

### 3.4 练习生成

选择：

```text
日语老师
Level: N5
```

点击「选择题」快捷按钮，或输入：

```text
请按 N5 水平出 5 道日语选择题，主题是点餐。
```

通过标准：
- 页面出现练习卡片。
- 卡片显示题目数量、主题和水平。
- 选择题选项可点击。
- 点击「查看解析」可看到答案和中文解析。
- 题干旁朗读按钮能播放目标语音频。

## 4. 回归检查

### 4.1 默认聊天页不受影响

访问：

```text
http://localhost:5173/
```

输入：

```text
你好
```

通过标准：
- 默认 Asuka 助手正常流式回复。
- Live2D 表情标签不会显示在聊天文本中。
- 不应出现语言学习卡片，除非后端明确下发语言工具结果。

### 4.2 Wiki 工具隔离

在语言学习页输入：

```text
帮我生成 wiki
```

通过标准：
- 语言老师不应调用 `generate_wiki`。
- 如果回复无法执行，也应以普通教学对话方式解释，而不是执行 wiki 工具。

## 6. 暂不验收项

- WebRTC 实时语音模式：本阶段只预留入口，按钮应为禁用状态。
- STT 跟读评分。
- 跨会话错误记忆。
- 生词卡与 SRS。
